Отправка перевода по почте
Это раздел позволяет Вам отослать обновленный перевод по почте.

Чтобы правильно отсылать сообщения Вы должны перейти в раздел 'Настройка модуля' Переводчика модулей Webmin и заполнить следующие данные:

  • Ваш почтовый ящик, который необходим для связи с Вами если возникнут какие-либо вопросы относительно Вашего перевода.
  • Сервер SMTP, через который будет осуществляться отправка сообщений.
  • Почтовый ящик команды переводчиков для отправки официальных обновлений переводов Команде Webmin

В большинстве случаев Вам достаточно установить "Сервер SMTP" в 'localhost', если Вы используете UNIX систему с установленным SMTP сервером. Авторизация SMTP на данный момент не доступна.

Чтобы отправить официальное обновление перевода Вы должны указать адрес получателя <translations@webmin.com>, тогда Ваша работа будет добавлена в официальный дистрибутив Webmin. Если Вы не укажите настоящий ящик в поле "Ваш почтовый ящик" - Ваше сообщение не будет рассмотрено. Если оставить поле пустым - адресом получателя по умолчанию будет являться <translations@webmin.com>.

В большинстве случаев Вам нужно отослать перевод только для Вашего языка, а не для всех. Пожалуйста, при отправке выберите Ваш язык из списка.

Важное замечание: Некоторые дистрибутивы для Linux (например, Mandrake) поставляются с большим набором модулей Webmin, которые не включены в основной дистрибутив Webmin.Пожалуйста, отсылайте Ваш перевод разработчикам этих модулей напрямую..
Полный список стандартных модулей Webmin можно найти здесь:
http://www.webmin.com/standard.html
Если у Вас есть какие-либо вопросы - напишите нам на почтовый адрес <translations@webmin.com> до отправки Вашего перевода, и укажите где Вы нашли этот модуль, чтобы мы могли Вам помочь.
