Web Interface translation

This section helps you to translate lang/ and ulang/ files.

The main list displays the translations corresponding to the selected module.

  • The column Added lets you know if some present items in the default version do not exist in the translation.
  • The column Removed enables you to detect items which are in the translation but do not exist anymore in the default version. These items will be automatically removed if you update your language.

If you want to add a translation for a new language, please go to the Admin section.

To allow detection of modified items, besides added/deleted ones, you will have to build a fingerprint of the reference file after each updating of your translation.

Whenever the table reports any modified items:

  1. First click on the Edit-Icon for your language.
  2. Choose View changed items.
  3. Translate these changes.
  4. After this rebuild the fingerprint.

Hint: If you do the fingerprint before translating the changes, the changes will not be detected anymore.
