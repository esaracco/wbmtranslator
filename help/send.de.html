Versende E-Mail
Diese Sektion hilft Ihnen eine E-Mail mit den neuesten Übersetzungen zu versenden oder diese für späteren Versand herunter zu laden.

Um diese Sektion nutzen zu können müssen Sie folgende Angaben in der 'Modulkonfiguration' unbedingt ausfüllen:

  • Ihre E-Mail-Adresse, welche für eine gültige Absenderadresse erforderlich ist, falls jemand Rückfragen hat.
  • SMTP-Server, damit Sie überhaupt eine E-Mail versenden können.
  • Webmin-Übersetzungs-Team E-Mail-Adresse um Ihre Updates offiziell einzureichen

Meistens möchten Sie den "SMTP-Server" auf 'localhost' einstellen, wenn Sie sich auf einem unixoiden System befinden. Damit sind Sie unabhängig vom installierten Mailserver. SMTP-Authentifizierung ist derzeit nicht möglich.

Um Updates offiziell einzureichen müssen Sie als Empfänger-Adresse unbedingt <translations@webmin.com> einstellen, so daß Ihre Updates für die Webmin-Software berücksichtigt werden kann. Wenn Sie keine gültige Absender-Adresse einstellen, dann wird Ihre Arbeit abgelehnt. Wenn Sie hier nichts engeben, dann wird <translations@webmin.com> als Standardwert eingetragen.

Ebenso möchten Sie sicherlich kein Update für alle Sprachen einsenden, sondern nur für eine. Bitte wählen Sie hier die richtige Sprache aus.

Wichtiger Hinweis: Je nach Distribution wurden mehr Module in die Webmin-Software eingebunden, als die eigentlich Webmin-Software hat. Updates für Nicht-Original-Webmin-Module reichen Sie bitte beim entsprechenden Maintainer ein!
Eine komplette Liste der Webmin-Standard-Module kann hier eingesehen werden:
http://www.webmin.com/standard.html
Wenn Sie immer noch unsicher sind, dann kontaktieren Sie <translations@webmin.com> bevor Sie ein Update einsenden möchten und geben Sie Hinweise, wo Sie das Modul gefunden haben. Auf diese Weise kann Ihnen besser geholfen werden.
