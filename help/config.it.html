Traduzione di "config.info"
In questa sezione è possibile tradurre le stringhe contenute nel file "config.info" nella propria lingua.

Questo file contiene quanto viene mostrato facendo clic su "Configurazione del modulo" per configurare un modulo.

In pratica questa sezione modifica il file seguente:
<webmininstalldir>/<module>/config.info.xx
dove "xx" rappresenta il codice della propria lingua.
