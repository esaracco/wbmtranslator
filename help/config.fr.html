Traduction de 'config.info'
Cette section vous permet de traduire le fichier 'config.info' dans votre langue.
Il s'agit de la traduction des textes que vous voyez lorsque vous cliquez sur le lien 'Configuration du Module' en vue de configurer celui-ci.

Techniquement, cette section édite le fichier suivant:
webmininstalldir/module/config.info.xx
ou 'xx' représente le code de votre langue..
