Traduction de la configuration

Cette section vous aide à convertir le fichier config.info.

La liste principale affiche les traductions correspondant à l'écran de configuration du module sélectionné.

      
  • La colonne Ajouté vous permet de savoir si certains éléments présents dans le fichier config.info n'existent pas dans la traduction.
    •   
  • La colonne Supprimé vous permet de détecter les éléments de la traduction qui n'existent plus dans config.info.

Pour permettre la détection des éléments modifiés, en plus des éléments ajoutés / supprimés, vous devrez créer une empreinte du fichier de référence après chaque mise à jour de votre traduction.

Chaque fois que le tableau rapporte des éléments modifiés:

      
  1. Premièrement cliquez sur l'icône d'édition pour votre langue.
    2.   
  2. Choisissez Afficher les éléments modifiés.
    3.   
  3. Traduisez les modifications.
    4.   
  4. Après cela, reconstruisez l'empreinte.

Conseil: si vous créez l'empreinte avant de traduire les modifications, celles-ci ne seront plus détectées.
