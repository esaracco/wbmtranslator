Share translations via E-Mail

This section helps you to send out an E-Mail with the updated translation or lets you download your work for later usage.

In order to make proper use of this section you'll need to edit the configuration of wbmtranslator with the following data:

  • Your E-Mail-Adress which is needed to get a valid sender adress in case someone needs to contact you regarding your work.
  • SMTP Host which is needed to send out an E-Mail anyway.
  • Webmin traduction team E-Mail-Adress to send in official updates to the Webmin-Core-Team

Most of the time you want to set SMTP Host to localhost if you are on a POSIX system regardless what SMTP-Server is installed on your system. SMTP-Authentification is not possible at this time.

In order to send in updates officially you want to set the recipient address to <translations@webmin.com> so that your work can be put into the main Webmin-Software. Be prepared that if you do not use a valid entry in Your E-Mail your updates are declined. If you leave this line blank this module defaults to <translations@webmin.com>.

Most of the times you only want to send in updates for your language and not for all. Please pick the right one from the list.
